# Multi Agent Data Research System (MADRS)

A multi-agent data research system is a system where multiple specialized software agents cooperate to collect, process, analyze, and synthesize information from many sources. Each agent performs a narrow role, and coordination produces a higher-quality research output than a single monolithic model.
